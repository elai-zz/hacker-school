This is Mocky.

Mocky is a small JavaScript library with no dependencies. Mocky allows you to accomplish pattern matching found in functional programming languages such as ML, Haskell, Erlang, and OCaml in JavaScript. Currently, Mocky is capable of pattern matching with one parameter (see below examples). There are plans in the future to expand, or feel free to contribute. Mocky uses Jasmine testing framework and another library similar to Mocky is Funcy.

How to use?

Simply include mocky.js in your .html file.

Examples

  1. Make a new Mocky object and load object with patterns. For anObj (below), if you call anObj(2), you'll get 3. If you call anObj(3), you'll invoke the anonymous function that would call some function foo() and return "b1" as a result from "b" + 1. 
var anObj = mocky(
    [2,3], 
    [1,2],
    [3 , function() {
        a = foo();
        return "b" + 1}],
    ["apple", function (n) { return "I like " + n}],
    [_, "wow!"]
);
